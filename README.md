# ClgMetta - College Discovery Platform

A comprehensive college discovery platform with seat booking, exam information, course details, tuition recommendations, and career guidance.

## Features

1. **Career Guidance System** - Interactive Q&A assessment to find perfect career paths
2. **Dynamic College Details** - Each college shows specific information with seat booking
3. **State-wise Exams** - Detailed exam information with patterns and syllabus
4. **Course Catalog** - Comprehensive course details with career opportunities
5. **Tuition Center Finder** - Online/Offline coaching centers with batch information
6. **College Predictor** - AI-powered college prediction based on rank
7. **Login System** - Complete authentication with validation and social login
8. **Interactive Maps** - Location maps for colleges and tuition centers
